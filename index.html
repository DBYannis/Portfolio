Qui-suis-je ?

Je code depuis Juin 2021 grâce à CodeCademy où j’ai effectué la carrière Front-End. J’ai ensuite suivi un bootcamp à La Capsule afin de décrocher mon diplôme de développeur Full-Stack.

J’ai toujours été curieux de comprendre comment fonctionnait les différentes technologies, c’est pourquoi le développement web m’a beaucoup plu. Je me suis d’abord tourné vers le front-end car j’aime le côté artistique et créatif qui permet de donner vie à des projets. Cependant le back-end m’a permis de mieux comprendre comment fonctionnait ce que j’avais appris en front-end.

Quand je ne code pas, je développe mes autres passions telles que le cinéma, les jeux vidéos et jeux de société, la musique et l’automobile.

Mes Compétences

J’utilise HTML, CSS, JavaScript ainsi que React-Redux pour le front-End. Node, Express et MongoDB sont des outils que j’utilise pour le back-end.

Ce que préfère dans le développement web est le front-end car j’aime créer et designer des applications claires et ergonomiques autant dans l’UI que l’UX.

Durant ma formation j’ai du gérer une équipe de trois autres personnes afin de délivrer une application mobile. Je suis donc capable de travailler en équipe et de surmonter les difficultés avec mes collègues

Mes Projets

DrugSync

DrugSync est une application que nous avons créée en tant que projet de fin de formation à La Capsule. Cette application mobile est déstinée aux médecins afin de les aider à la prescription de médicament en fonction du traitement et des pathologies de ses patients. Lorsque le médecin sélectionne la fiche patient préalablement enrengistrée il peut faire des recherche de médicatment, mollécule ou pathologie. L'application indiquera ensuite si le traitement que le médecin a recherché entre en conflit avec son traitement et sa ou ses pathologies.

Music Store

Music Store est une simple application faite en React afin de recréer un site de vente d'enceintes audio.

My Moviz

My Moviz est une application utilisant un web service qui permet d'afficher les derniers films sortis avec leurs notes respectives. Il est possible de les noter sur 10, de les marquer comme 'vu' et d'ajouter un 'like' afin de les sauvegarder dans le menu en haut à gauche où il est également possible de les supprimer.

React Pokédex

React Pokédex est une application en react qui répertorie les pokémon en utilisant une api (PokéAPI) qui récupère les données des pokémon comme leur nom, image, et type.
